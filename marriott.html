Marriott Homes & Villas Website Redesign

Global Career Accelerator + Publicis Sapient | August 2024

Project Overview

This project was part of a web development and prototyping case study in partnership with Publicis Sapient. The challenge was to redesign Marriott's Homes & Villas booking site to improve the user experience, accessibility, and feature prioritization for travelers booking vacation rentals.

My Role

  • UX/UI Designer
  • Researcher
  • Front-End Developer

Problem Statement

The original Homes & Villas experience lacked personalization and clarity in navigation, especially when users were filtering listings or comparing properties. The site also lacked consistent accessibility standards and visual hierarchy in key decision-making flows.

Research

  • Analyzed qualitative user interviews and quantitative surveys to identify common frustrations with the booking process
  • Reviewed competitive platforms such as Airbnb and VRBO for usability insights and feature benchmarking
  • Mapped key user journeys (e.g., search → compare → book) to uncover friction points

Insights

  • Users want a simplified filtering experience and visual cues that help distinguish listings quickly
  • Accessibility and mobile responsiveness were inconsistent across devices
  • Users valued transparency in pricing and amenities more than flashy visuals

Design & Prototyping

  • Created a high-fidelity prototype using Figma, featuring simplified navigation, improved listing cards, and clearer CTAs
  • Designed components for a custom design system (buttons, inputs, filters, listing tiles)
  • Prioritized accessibility with consistent spacing, contrast, and readable typography

Development

  • Built the redesigned homepage using HTML, CSS, and JavaScript
  • Implemented responsive layout with semantic HTML and Bootstrap
  • Ensured the structure met accessibility standards and passed visual contrast checks
