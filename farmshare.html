Farmshare Austin Website Redesign

Accessibility Internet Rally | August–November 2024

Overview

As part of the Accessibility Internet Rally hosted by Knowbility, I led a team in redesigning the nonprofit Farmshare Austin’s website to improve accessibility, user experience, and content clarity using WordPress and custom HTML/CSS.

My Role

  • Team Lead & UX Designer
  • Client Communication
  • Usability Testing
  • Front-End Development

Problem Statement

The original site lacked structure, had outdated content, and failed WCAG accessibility standards, making it hard to use for people relying on assistive technologies.

Process

  • Audited content and accessibility using tools like WAVE
  • Built wireframes and mockups based on user needs
  • Developed a WordPress site using semantic HTML and custom CSS
  • Conducted usability testing and iterated based on results

Final Designs

The final website passed WCAG accessibility checks and created a smoother, more intuitive experience for Farmshare’s audience.

Reflection

This project taught me the importance of inclusive design and how accessibility considerations can improve the experience for all users. It also gave me confidence in leading a team and communicating with a real nonprofit client.
